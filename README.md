# Anti-Logger
In efforts to stop malware to a point.

## Request
If you want to add a logger to this please fork this and send in a pull request. I will review it and decide to merge or not.

* Cookie Ghost
* Cookie Venom
* Sir Cookie
* browserLoot
* njRAT
* QuasarRAT
* Cookie Muncher
* Predator Logger
* Vulcan Logger
* NingaliNET
* NanoCore
* Comet RAT
* Revenge RAT
* XStealer
